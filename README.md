📚 Study Buddy AI Chatbot

Description

Study Buddy AI Chatbot is a rule-based chatbot developed using Python. It interacts with users through predefined responses and helps them with basic AI information, study tips, motivation, and time-related queries.

The chatbot uses if-else statements and a continuous loop to simulate simple human conversation.

Features

- Greets users with friendly messages
- Provides information about Artificial Intelligence
- Gives useful study tips
- Motivates users with positive quotes
- Shows the current time
- Handles exit commands gracefully
- Runs continuously until the user exits

Technologies Used

- Python 3
- GitHub
- Replit

How to Run

1. Open the project in Replit or any Python IDE.
2. Run the file "main.py".
3. Type your message and press Enter.
4. Type "exit", "bye", or "quit" to close the chatbot.

Sample Conversation

You: hello

Bot: Hello! Ready to learn something new today?

You: study tip

Bot: Study for 25 minutes and take a 5-minute break. This is called the Pomodoro Technique.

You: motivate me

Bot: Success is the sum of small efforts repeated every day. Keep going!

You: bye

Bot: Goodbye! Best of luck with your studies.
